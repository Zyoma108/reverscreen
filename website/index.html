REVERSCREEN COMPUTER-VISION SNIPPING TOOL

REVERSCREEN is a snipping tool that leverages smartest Computer-Vision algorithms to simplify precise objects capturing.
QUICK START GUIDE
Select approximate area of interest (see solid red line)
A recognized object is highlighted with dashed red line
Recognized object is automatically selected - just click Crop.
Select another (sub)object, precisely - with Ctrl button.
Optionally, use Colors mode to pick any colors from the image.
Click Copy or Save buttons to export your snipping.
